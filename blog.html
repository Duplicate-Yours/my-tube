Discuss the scope of `var`, `let`, and `const`.

In this article we have seen that var declarations are globally scoped or function scoped while let and const are block-scoped. var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared. They are all hoisted to the top of their scope.

What do you mean by REST API?

A REST API (Representational State Transfer Application Programming Interface) is an architectural approach for designing web services that are stateless, use standard HTTP methods (GET, POST, PUT, DELETE, etc.) to perform operations on resources identified by URLs, and follow a uniform and consistent interface for communication between clients and servers. It is a widely used method for building scalable and interoperable web-based systems.
